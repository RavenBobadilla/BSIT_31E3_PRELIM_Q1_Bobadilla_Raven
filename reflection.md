# Reflection Answers

# 1. Why did you use inheritance?

I used inheritance because all the transports are vehicles. It helps make the code cleaner and easier to organize since they all share the same base class.

# 2. Why did you use interfaces?

I used interfaces because they represent abilities. For example, cars can drive, planes can fly, and boats can sail.

# 3. Can Helicopter inherit from both Vehicle and Airplane? Why or why not?

No, because C# only allows one parent class. A Helicopter can inherit only from Vehicle.

# 4. Why can Helicopter implement both IFlyable and IDriveable?

Because C# allows multiple interfaces. A helicopter can both fly and drive, so it can implement both interfaces.

# 5. If a Submarine can both sail and dive, how would you design it?

I would make Submarine inherit from Vehicle and implement both ISailable and IDiveable interfaces.
